# GitHub Copilot Instructions
## Suggested Prompts for Each File

Use these prompts when working on specific files. GitHub Copilot will help you fill them out.

### 01_Career_Master.md

- "Help me quantify the impact of [PROJECT NAME]"
- "Expand on responsibilities for a [YOUR TITLE]"
- "Write achievement bullets for [ACCOMPLISHMENT]"

- "Help me categorize my skills by proficiency"

## General Tips

1. **Be specific:** Include your role, industry, or project name
2. **Iterate:** Start broad, then ask for expansion
3. **Context matters:** Reference other files when relevant
4. **Personalize:** Copilot's suggestions are starting points - make them yours

## Example Workflow

1. Open a template file
2. Read the existing structure
3. Type a Copilot prompt (or start writing naturally)
4. Accept suggestions, edit as needed
5. Link to related files
6. Repeat for other sections

**Copilot works best when you:**
- Provide context about your role/industry
- Use natural language prompts
- Edit and refine suggestions
- Reference your actual experiences
