# Quick Start Guide
## Get Your Knowledge Engine Running in 5 Minutes

## Step 1: Navigate to the Kernel

```bash
cd modules/kernel/aurelion-kernel-lite
```

## Step 2: Open in VS Code

```bash
code .
```

## Step 3: Start with the Hub

Open `Floor_01_Foundation/00_Hub_Index.md`

This is your central navigation. It links to all other files.

## Step 4: Fill Your Career Master

Open `Floor_01_Foundation/01_Career_Master.md`

Start typing your current role. GitHub Copilot will suggest:
- Responsibilities
- Achievements
- Skills

**Try this prompt:**  
"Help me document my career as a [YOUR ROLE] at [COMPANY]"

## Step 5: Build Your Skills Inventory

Open `Floor_01_Foundation/02_Skills_Inventory.md`

List your skills. Copilot will help categorize them.

## Step 6: Explore the Other Floors

- **Floor 02:** Add your terminology and standards
- **Floor 03:** Map your professional network
- **Floor 04:** Use project templates for active work
- **Floor 05:** Complete your personality framework and strategic plan

## Step 7: Customize

This is YOUR knowledge base. Add files, remove files, restructure as needed.

## Optional: Install aurelion-memory-lite

For advanced knowledge graph features:

```bash
pip install aurelion-memory-lite
```

## Tips

1. **Use GitHub Copilot** - It's trained on this structure
2. **Link between files** - Use relative markdown links
3. **Keep it current** - Update regularly (weekly review)
4. **Make it yours** - Don't feel constrained by templates
